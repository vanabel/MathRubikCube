# 🎮 交互式GUI使用指南

## 启动方式

```bash
python visualize.py -i
```

或

```bash
python visualize.py --interactive
```

## 🎯 界面说明

### 布局

```
┌────────────────────────────────────────────────┐
│  图例              魔方展开图              统计  │
│                                                │
│  (左上角)        (中央显示区)         (右上角) │
│                                                │
│              标题：应用的公式                   │
│              角块循环 / 棱块循环                │
├────────────────────────────────────────────────┤
│ 公式: [输入框___________] [应用][重置][撤销]   │
├────────────────────────────────────────────────┤
│ 历史: F → R U → [F,R] → ...                  │
└────────────────────────────────────────────────┘
```

### 组件说明

**1. 图例（左上角）**
- 🟢 角块未动
- 🔴 角块已动
- 🔵 棱块未动
- 🟠 棱块已动
- 说明：U/L/F/R/B/D为面中心（固定）

**2. 魔方展开图（中央）**
- 展开的六个面
- 块上显示编号或"位置→块"
- 鼠标悬停显示详细信息 ✨

**3. 统计信息（右上角）**
- 移动的角块数量
- 移动的棱块数量

**4. 输入框**
- 输入公式（如 F, R U, [F,R]）
- 按Enter或点击"应用"

**5. 按钮**
- **应用**: 执行输入的公式
- **重置**: 回到初始状态
- **撤销**: 回退上一步

**6. 历史记录（底部）**
- 显示最近5个操作

## 🖱️ 鼠标悬停功能

### 如何使用

将鼠标移到任意块上，会弹出黄色提示框，显示：

**角块示例：**
```
位置: UFR (角块0)
当前块: DFR (角块4)
状态: 已移动
```

**棱块示例：**
```
位置: UF (棱块0)
当前块: UF (棱块0)
状态: 未移动
```

**面中心：**
```
面中心: F
```

### 故障排除

如果鼠标悬停不工作：

**方法1: 检查matplotlib后端**
```python
import matplotlib
print(matplotlib.get_backend())
```

如果不是交互式后端（如TkAgg、Qt5Agg），可以设置：
```python
# 在代码开头添加
import matplotlib
matplotlib.use('TkAgg')  # 或 'Qt5Agg', 'MacOSX'
```

**方法2: 测试悬停功能**
```bash
python test_hover.py
```

这会打开一个简单的测试窗口，如果这个测试也不工作，说明是matplotlib环境问题。

**方法3: 重新安装matplotlib**
```bash
pip uninstall matplotlib
pip install matplotlib
```

## 💡 使用技巧

### 1. 探索交换子

```
输入: [F,R]
按Enter
观察: 角块和棱块如何移动

输入: [R,U]
按Enter
观察: 不同的移动模式

输入: [F,[R,U]]
按Enter
观察: 嵌套交换子的效果
```

### 2. 构建复杂序列

```
步骤1: F
步骤2: R U
步骤3: F'
查看累积效果
```

### 3. 实验和撤销

```
输入随意的公式
如果不喜欢，点击"撤销"
再尝试其他公式
```

### 4. 查找三循环

```
输入: [F,[R,U]]
查看标题: 角块: (0 4 3) ← 这是三循环！
```

## 🎨 快捷键

- **Enter**: 应用公式
- 鼠标悬停在块上: 显示详细信息

## 📝 支持的公式

### 基本操作
- `F`, `R`, `U` - 顺时针90度
- `F'`, `R'`, `U'` - 逆时针90度
- `F2`, `R2`, `U2` - 180度

### 序列
- `F R U` - 连续执行
- `R U R' U'` - Sexy Move
- `R U R' U R U2 R'` - Sune

### 交换子
- `[F,R]` - F R F' R'
- `[R,U]` - R U R' U'
- `[F,[R,U]]` - 嵌套交换子

## 🐛 常见问题

### Q: 鼠标悬停不显示
**A:** 
1. 检查matplotlib后端（见上文）
2. 运行 `python test_hover.py` 测试
3. 确保鼠标在彩色块上，不是空白处

### Q: 输入公式后没反应
**A:** 
1. 检查终端是否有错误信息
2. 确保公式语法正确
3. 点击"应用"按钮或按Enter

### Q: 窗口太大/太小
**A:** 
修改 `visualize.py` 中的：
```python
fig = plt.figure(figsize=(12, 9))  # 调整这里的尺寸
```

### Q: 中文显示乱码
**A:** 
已经配置了中文字体，如果还有问题，参考 `FONT_FIX.md`
