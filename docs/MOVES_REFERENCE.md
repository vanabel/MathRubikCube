# 🎲 魔方操作快速参考

## 📊 完整操作列表（18种）

### 基本操作（6个）

| 操作 | 名称 | 描述 | 角块循环 | 棱块循环 |
|------|------|------|----------|----------|
| **F** | Front 前 | 前面顺时针90° | (0 4 5 1) | (0 4 8 5) |
| **R** | Right 右 | 右面顺时针90° | (0 3 7 4) | (3 7 11 4) |
| **U** | Up 上 | 上面顺时针90° | (0 3 2 1) | (0 3 2 1) |
| **L** | Left 左 | 左面顺时针90° | (1 2 6 5) | (1 6 9 5) |
| **B** | Back 后 | 后面顺时针90° | (2 6 7 3) | (2 6 10 7) |
| **D** | Down 下 | 下面顺时针90° | (4 5 6 7) | (8 9 10 11) |

### 逆操作（6个）

| 操作 | 描述 | 等价于 |
|------|------|--------|
| **F'** | 前面逆时针90° | F F F 或 F³ |
| **R'** | 右面逆时针90° | R R R 或 R³ |
| **U'** | 上面逆时针90° | U U U 或 U³ |
| **L'** | 左面逆时针90° | L L L 或 L³ |
| **B'** | 后面逆时针90° | B B B 或 B³ |
| **D'** | 下面逆时针90° | D D D 或 D³ |

### 双层旋转（6个）

| 操作 | 描述 | 等价于 |
|------|------|--------|
| **F2** | 前面180° | F F |
| **R2** | 右面180° | R R |
| **U2** | 上面180° | U U |
| **L2** | 左面180° | L L |
| **B2** | 后面180° | B B |
| **D2** | 下面180° | D D |

## 📐 魔方记法

### 标准记法

```
    UBL---UBR
    /|    /|
  UFL---UFR|
   |DBL-|-DBR
   |/   |/
  DFL---DFR
```

### 面的标记

- **U** (Up) - 上面，🟨 黄色
- **D** (Down) - 下面，⚪ 白色
- **F** (Front) - 前面，🟩 绿色（面向你）
- **B** (Back) - 后面，🟦 蓝色
- **L** (Left) - 左面，🟥 **红色**
- **R** (Right) - 右面，🟧 **橙色**

**注意：** 本项目中L面=红色，R面=橙色（这是标准配色）

### 操作的方向

- **顺时针**：从该面看是顺时针（无后缀）
- **逆时针**：从该面看是逆时针（加 `'`）
- **180度**：旋转两次（加 `2`）

## 🎯 常用公式

### Sexy Move
```
R U R' U'
角块: (0 3) (2 7)
棱块: (2 3 7)
```

### Sune (OLL)
```
R U R' U R U2 R'
角块: (0 7) (1 2)
棱块: (1 2 3)
```

### T-Perm (PLL)
```
R U R' U' R' F R2 U' R' U' R U R' F'
角块: (0 3) (2 5) (4 7)
棱块: (1 3) (4 11 8)
```

### Y-Perm (PLL)
```
F R U' R' U' R U R' F' R U R' U' R' F R F'
角块: (0 2) (3 4) (5 7)
棱块: (1 2) (4 11 8)
```

### Sledgehammer
```
R' F R F'
角块: (0 1 3)
棱块: (0 4 3)
```

### Hedgeslammer
```
F R' F' R
角块: (0 3 1)
棱块: (0 3 4)
```

## 🧮 群论性质

### 基本性质

- **X⁴ = e**：任意基本操作旋转4次回到初始状态
- **X · X' = e**：操作与其逆复合等于恒等
- **X² · X² = e**：双层旋转两次回到初始状态
- **[A, B]**：交换子，通常产生三循环

### 魔方群

- **生成元**：{F, R, U, L, B, D}
- **群阶**：约 4.3 × 10¹⁹
- **结构**：S₈ × S₁₂ 的子群（加上朝向约束）

### 交换子性质

- **[A, B] = ABA⁻¹B⁻¹**
- 交换子是偶置换
- 常产生三循环
- 可以嵌套：[A, [B, C]]

## 💡 使用技巧

### 1. 快速测试操作

```python
from cube import *

# 查看某个操作的效果
analyze_algorithm("L")
analyze_algorithm("B D")
```

### 2. 探索交换子

```python
# 所有简单交换子
for a in ["F", "R", "U", "L", "B", "D"]:
    for b in ["F", "R", "U", "L", "B", "D"]:
        if a < b:  # 避免重复
            analyze_algorithm(f"[{a}, {b}]")
```

### 3. 在GUI中实验

```bash
python visualize.py -i

# 然后尝试：
# L
# B D
# [L, D]
# [B, [L, D]]
```

### 4. 构造特定循环

寻找纯三循环（只移动3个块）：

```python
from cube import *

# 嵌套交换子通常产生纯三循环
test_algs = [
    "[F, [R, U]]",
    "[L, [B, D]]",
    "[R, [F, L]]",
]

for alg in test_algs:
    corner, edge = get_algorithm_permutation(alg)
    c_cycles = permutation_cycles(corner)
    e_cycles = permutation_cycles(edge)
    
    print(f"{alg:15s} 角块: {format_cycles(c_cycles)}")
```
