## 🎨 核心特性展示

### 1. 交换子计算

```python
analyze_algorithm("[F, [R, U]]")
# 输出：
# 展开为: F R U R' U' F' U R U' R'
# 角块循环: (0 4 3)  ← 三循环！
# 棱块循环: identity
```

### 2. 著名公式分析

支持分析：
- Sexy Move: `R U R' U'`
- Sune: `R U R' U R U2 R'`
- T-Perm, J-Perm 等 PLL 公式

### 3. 公式阶计算

```python
# Sexy Move 的阶 = 6
order("R U R' U'")
```

### 4. 三循环探索

自动识别产生三循环的公式，对魔方还原算法研究很有用。

## 💡 技术亮点

### 1. 优雅的置换表示

使用列表索引表示置换，直观且高效：

```python
F_CORNERS = [4, 0, 2, 3, 5, 1, 6, 7]
# 位置i的块来自F_CORNERS[i]
```

### 2. 递归的交换子展开

支持任意层次的嵌套：

```python
"[F, [R, U]]" → "F R U R' U' F' U R U' R'"
```

### 3. 循环分解算法

O(n) 时间复杂度的循环识别：

```python
permutation_cycles([1, 2, 0, 3]) → [(0, 1, 2)]
```
