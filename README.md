# 🎲 魔方置换群数学模型

一个用Python实现的三阶魔方（Rubik's Cube）置换群数学模型，支持公式解析、交换子计算和循环分解分析。

## 🎯 项目目标

1. 用数学上严谨的方式描述三阶魔方的置换结构
2. 对任意公式计算其作用下的块置换
3. 支持交换子和嵌套交换子计算
4. 输出置换的循环分解
5. 后续可扩展至可视化和群论分析

## 📐 数学模型

### 块编号系统

**角块编号（0-7）：**

```
    2---3
   /|  /|
  1---0 |    上层 (U)
  | 6-|-7
  |/  |/
  5---4        下层 (D)
```

- 0: UFR (上前右)
- 1: UFL (上前左)
- 2: UBL (上后左)
- 3: UBR (上后右)
- 4: DFR (下前右)
- 5: DFL (下前左)
- 6: DBL (下后左)
- 7: DBR (下后右)

**棱块编号（0-11）：**

- 0-3: 上层 (UF, UL, UB, UR)
- 4-7: 中层 (FR, FL, BL, BR)
- 8-11: 下层 (DF, DL, DB, DR)

### 置换表示

每个操作定义为两个置换：

- `corner_perm`: 长度为8的列表，表示角块置换
- `edge_perm`: 长度为12的列表，表示棱块置换

例如，F（前面顺时针90度）：

- 角块循环：(0 4 5 1)
- 棱块循环：(0 4 8 5)

## 🔧 公式语法

### 基本操作

- `F`, `R`, `U`: 前、右、上面顺时针90度
- `F'`, `R'`, `U'`: 逆时针90度
- `F2`, `R2`, `U2`: 180度旋转

### 交换子

- `[A, B]`: 交换子，等价于 `A B A' B'`
- `[F, [R, U]]`: 支持嵌套

### 公式示例

```python
# 简单公式
analyze_algorithm("F R U R' U' F'")

# 交换子
analyze_algorithm("[R, U]")

# 嵌套交换子
analyze_algorithm("[F, [R, U]]")

# Sune算法
analyze_algorithm("R U R' U R U2 R'")
```

## 🧮 群论背景

魔方群是一个巨大的有限群：

- 群阶：约 4.3 × 10¹⁹
- 生成元：{F, R, U, L, B, D}
- 结构：S₈ × S₁₂ 的子群（加上朝向约束）

交换子 `[A, B] = ABA⁻¹B⁻¹` 是群论中的重要构造，用于：

- 研究换向子子群
- 构造特定的块循环
- 分析算法的代数结构
